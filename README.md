## 🔥 Fitur Aplikasi  
✅ **Deteksi Penyakit** pada tanaman sawi menggunakan **CNN**  
✅ **User-friendly UI** dengan tampilan yang minimalis  
✅ **Aplikasi berbasis Android** yang ringan dan mudah digunakan  
✅ **Database Penyakit** untuk melihat informasi penyakit pada tanaman

💻 **Platform:** Android  
📲 **Framework:** TensorFlow Lite
